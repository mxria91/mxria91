  • Wie wird eine Variable deklariert?

Eine Variable wird mit var + Zuweisungsoperator (=) und dem Wert deklariert. Beispiel:
var name = "Maria";

  • Wie wird eine Konstante deklariert?

Eine Konstante wird genau wie bei var deklariert, mit dem Unterschied, dass const statt var verwendet wird.
Beispiel: const name = "Maria";

  • Wo liegt der Unterschied zwischen var, let und const?

"var" ist die veraltete Schreibweise, um eine Variable zu deklarieren. Der Keyword "var" kann global verwendet werden (global scope), sprich auch außerhalb einer Blocks und der Wert dieser Variable lässt sich immer neu zuweisen. Seit ES6 wurden die Variablen let und const eingeführt. "let" ist ähnlich wie var, der Wert dieser Variable kann ebenfalls neu zugewiesen werden, bei const (eine Konstante) hingegen nicht. Weiters sind die beiden neuen Variabeln let und const sogenannte Block-Scope Variabeln. Sie lassen sich nur innerhalb eines Blocks verwenden.

  • Was ist ein type und welche gibt es?

Es gibt Strings, Booleans, Integer, undefined und null. Das sind die sogenannenten Primitiven Datentypen. Dann gibt es die komplexe Datentypen wie Funktionen, Arrays und Objekte.

  • Wie kann ein Datentyp geprüft werden?

Mit typeOf in der Konsole des Browsers. Beispiel:
console.log (type of var);

  • Was ist der Unterschied zwischen undefined und not defined

Bei undefined wurde die Variable definiert (var name) aber keinem Wert zugewiesen, sprich ohne Zuweisungsoperator. Bei der anderen (not (yet) :P defined) wurde die Variable überhaupt nicht definiert.

  • Wo kann ich meinen JavaScript Code "ausprobieren" und debuggen?

Man kann entweder direkt im Browser den Code über einen Live-Server ausprobieren und ggf. debuggen oder man verwendet diverse Extensions / Features nutzen. Mit Hilfe von Breakpoints kann man Schritt für Schritt nachvollziehen, was mit dem Code passiert.

  • Wo im Browser können Werte ausgegeben werden?

In der Konsole in den Developer Tools.

  • Wie kann eine Zeichenkette mit einer weiteren verbunden werden?

Durch konkatenieren können mehrere Zeichenketten miteinander verbunden werden. Als Operator verwendet man +
Beispiel:
let vorname = "Maria";
let nachname = "Repolust";
const name = vorname + " " + nachname;

  • Wie wird ein 
    array
    in JavaScript deklariert?

Ein Array wird wie folgt deklariert: let name = [ "Maria", "Frankie", "Michael" ];

  • Wie kann ein Element in einem 
    array
    entfernt oder hinzugefügt werden?

Mit sogenannten Methoden wie pop, delete, slice oder push. Man kann bspw. den letzten oder den ersten Wert eines Arrays entfernen oder hinzufügen usw.
Beispiel:
const plants = ['broccoli', 'cauliflower', 'cabbage', 'kale', 'tomato'];
console.log(plants.pop());
// Output: "tomato" wird aus dem Array gelöscht.
